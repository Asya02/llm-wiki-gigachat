# Suggested live demo script

1. Show the raw folder and explain that it contains messy source material: logs, emails, CSV, JSON, interviews, old archive, contradictions, and a prompt-injection trap.
2. Run the agent with `prompts/demo_prompt_full.md`.
3. Open `wiki/index.md`.
4. Open `wiki/timeline.md` to show chronology reconstruction.
5. Open `wiki/hypotheses/k-17-beacon-hypothesis.md` to show synthesis.
6. Open `wiki/contradictions.md` to show superseded claims.
7. Open `wiki/answers/final-explanation.md` to show the final answer.
8. Open `exports/llms.txt` to show the external LLM-readable artifact.
9. Optional: load the folder in Obsidian and show graph view.

Suggested narration:

"The interesting part is not that the model summarized files. It converted a messy raw dossier into a durable, cross-linked knowledge artifact with provenance, contradictions, hypotheses, and an exportable public explanation."
