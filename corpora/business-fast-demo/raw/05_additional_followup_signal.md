# Дополнительный сигнал после запуска базы

source_id: additional_followup_signal
date: 2026-05-16

Новый факт для демонстрации incremental ingest:
- На одном из других договоров найден похожий риск:
  после обновления тарифов снова не подтянулось исключение.
- Финансовой ошибки в счете еще нет, но есть предупреждение от мониторинга.

Решение:
- добавить pre-send проверку "есть ли договорные исключения" для всех крупных клиентов;
- включить алерт, если сумма счета изменилась более чем на 10% к прошлому месяцу.

Важно: этот файл специально подготовлен для второго шага демо ("добавим еще один файл"),
поэтому в базовом построении он НЕ используется.
