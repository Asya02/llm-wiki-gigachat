# Simple Bakery Business LLM-Wiki Demo Raw Pack

Это простой бизнесовый demo-pack для проверки LLM-Wiki подхода на понятной теме.

Сюжет: небольшая пекарня рядом с офисами думает, как за 2 недели увеличить утреннюю выручку без найма новых людей и без сложной доставки.

Главный вопрос:

> Какой простой бизнес-эксперимент стоит запустить пекарне: оставить всё как есть, добавить скидки вечером, или протестировать предзаказ офисных завтраков?

Цель демо — чтобы агент не просто пересказал файлы, а построил wiki: факты, цифры, отзывы, ограничения, риски, варианты решения и финальную рекомендацию.
