# Новые правила классификации

source_id: category_rules_new
status: current

Новое решение: классифицировать расход по цели покупки, если цель можно понять из данных.

Порядок приоритета:

1. Ручное исправление пользователя.
2. Заметка пользователя.
3. Описание покупки.
4. Справочник продавцов.
5. Старая банковская категория.

Правила:

- Если пользователь уже исправил похожую покупку, предложить такое же правило для будущих похожих покупок.
- Если описание содержит корм, наполнитель, игрушку для животного или когтеточку, категория «Питомец».
- Если описание содержит книгу, курс, тетрадь, блокнот для обучения или учебные материалы, категория «Обучение».
- Если описание содержит шампунь, крем, косметику, зубную пасту или солнцезащитное средство, категория «Уход за собой».
- Если описание содержит лекарства, врача, анализы или медицинское лечение, категория «Здоровье».
- Если описание содержит торт, свечи, одноразовую посуду, снеки для гостей или явно связано с друзьями, категория «Встречи и гости».
- Если описание содержит стиральный порошок, губки, лампочки, батарейки, контейнеры или скотч, категория «Дом и быт».
- Если данных недостаточно, категория не должна меняться автоматически. Нужно поставить статус «Нужно проверить».

Важно: система должна показывать простое объяснение, почему категория была изменена.
