# Профиль пользователя для демо

source_id: user_profile

Пользователь: синтетический демо-пользователь. Реальных персональных данных нет.

Пользователь хочет видеть категории не только по магазину, а по смыслу траты.

Желаемые категории:

- Еда домой — обычные продукты для дома.
- Обед на работе — кафе, доставка и готовая еда в рабочие дни.
- Встречи и гости — покупки для гостей, праздников, совместных вечеров и походов куда-то с друзьями.
- Дом и быт — уборка, лампочки, посуда, мелкие товары для квартиры.
- Питомец — корм, наполнитель, игрушки и ветеринарные расходы.
- Уход за собой — шампунь, крем, стрижка, косметика, солнцезащитные средства.
- Здоровье — лекарства, врач, анализы.
- Транспорт — такси, общественный транспорт, поездки.
- Обучение — книги, курсы, тетради, учебные материалы.
- Подарки — покупки, сделанные для другого человека.
- Подписки — регулярные цифровые сервисы.
- Прочее — всё, что не удалось уверенно отнести.

Особые предпочтения пользователя:

- Если есть ручное исправление, оно важнее старой банковской категории.
- Если есть заметка пользователя, она важнее типа магазина.
- Если категория неочевидна, лучше показать «Нужно проверить», чем уверенно ошибиться.
- Пользователь не хочет, чтобы прошлые категории менялись молча. Нужно показывать причину изменения.
