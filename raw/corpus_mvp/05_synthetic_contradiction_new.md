# New internal note

source_id: synthetic_contradiction_new

The current decision is to use AGENTS.md as the schema file for DeepAgents/Codex-style harnesses.
The previous CLAUDE.md-only rule is superseded as of 2025-06-01.
AGENTS.md provides better compatibility with multi-provider agent harnesses.
Both Claude Code (via CLAUDE.md) and DeepAgents (via AGENTS.md) are supported,
but the project default is now AGENTS.md.
